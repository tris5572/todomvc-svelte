# TodoMVC Svelte

よくあるサンプルアプリの TodoMVC を Svelte で書いてみたものです。

デモ： https://tris5572.github.io/todomvc-svelte/

## 使用技術

- TypeScript
- Svelte
- Vite
- pnpm
- Vitest

## 備考

よくある TodoMVC をそのまま実装したわけではなく、以下のような違いがあります。

- 表示中のアイテムをすべてトグルする機能はありません。（表示・挙動が自分の好みではないため）
- スタイルは独自のものです。

## ローカル起動

```sh
pnpm dev
```
