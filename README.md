# TodoMVC Svelte

[TodoMVC](https://todomvc.com/) を Svelte で書いてみたリポジトリです。

> [!NOTE]
> Svelte を自分の手に慣らすために書いたものなので、出来はそれなり程度です。

## 使用技術

- TypeScript
- Svelte
- Vite
- pnpm

## 備考

元の TodoMVC とは以下のような点で違いがあります。

- 表示中のアイテムをすべてトグルする機能はありません。（表示・挙動が自分の好みではないため）
- スタイルは独自のものです。（元のスタイルはレトロ感があるため）

## ローカル起動

```sh
pnpm dev
```
